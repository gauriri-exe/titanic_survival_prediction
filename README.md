# Titanic_survival_prediction
**Author:** Gauri Gupta  
**Date:** May 2026

### Project Objective
I built a machine learning model to predict whether a passenger survived the Titanic disaster or not.

This is my end-to-end project where I did:
- Data loading and exploration
- Data cleaning
- Feature engineering
- EDA with visualizations
- Model training (Logistic Regression + Random Forest)

**Best Accuracy:** 82.68% using Random Forest

## Conclusion & Learnings

This project helped me a lot in understanding the full data analysis process.

### What I Learned:
- How to clean real-world messy data (missing values, dropping columns)
- Importance of Feature Engineering (`Family_Size`, `Is_Alone`)
- How to do EDA and find insights (especially survival by gender and class)
- Training and comparing machine learning models
- The value of good visualizations

### Final Results:
- Logistic Regression Accuracy: **79.89%**
- **Random Forest Accuracy: 82.68%** 

Even though the notebook is a bit raw, it shows my actual learning journey — from struggling with basic functions to successfully building a working model.

**Future Improvements:** Hyperparameter tuning, trying XGBoost, and using the real `test.csv` file for Kaggle submission.
